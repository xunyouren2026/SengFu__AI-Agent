### 3.3 混合策略 (推荐)

```python
class HybridLLMStrategy:
    """
    智能路由策略:
    1. 简单任务 -> 本地模型 (免费+隐私)
    2. 复杂任务 -> 云端API (高性能)
    3. 敏感数据 -> 本地模型 (合规)
    4. 网络故障 -> 本地模型 (高可用)
    """
    
    def route(self, task_type, data_sensitivity, network_status):
        if data_sensitivity == "high":
            return self.local_model
        elif network_status == "offline":
            return self.local_model
        elif task_type == "complex":
            return self.cloud_api
        else:
            return self.local_model  # 默认本地，省钱
```
