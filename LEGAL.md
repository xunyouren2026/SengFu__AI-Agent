## 2. 隐私政策

### 2.1 数据收集范围

本软件**不收集任何用户数据**。具体而言：

- 本软件不设置任何远程数据采集、遥测（Telemetry）或用户行为追踪机制。
- 本软件不要求用户注册账户或提供个人信息。
- 本软件不使用Cookie、Web Beacon、像素标签或其他在线追踪技术。
- 本软件不收集用户的IP地址、设备信息、地理位置或其他标识信息。

### 2.2 本地存储说明

本软件在运行过程中会在用户本地设备上生成和存储以下数据：

| 数据类型 | 存储位置 | 说明 |
|---------|---------|------|
| 配置文件 | `~/.agi_framework/config/` | 包含API密钥、模型参数、系统设置等 |
| 运行日志 | `~/.agi_framework/logs/` | 记录系统运行状态、错误信息等 |
| 审计日志 | `~/.agi_framework/audit/` | 记录操作审计信息（含HMAC签名） |
| 任务历史 | `~/.agi_framework/history/` | 记录任务执行历史及结果 |
| 模型缓存 | `~/.agi_framework/cache/` | 缓存下载的模型文件及中间结果 |

上述数据完全存储在用户本地设备上，不会通过网络传输至任何第三方服务器。用户可随时查看、修改或删除上述数据。

### 2.3 API Key安全

1. **加密存储**: 本软件支持对API密钥进行加密存储（采用AES-256或同等强度的加密算法），以降低密钥泄露风险。
2. **不传输至第三方**: 本软件不会将用户的API密钥传输至除用户指定的API服务提供商之外的任何第三方。
3. **环境变量支持**: 用户可通过环境变量方式配置API密钥，避免将密钥明文写入配置文件。
4. **安全建议**: 建议用户定期轮换API密钥，使用独立的API子账号并设置合理的调用配额限制。

### 2.4 第三方服务

本软件可能调用以下类型的第三方服务，均由用户自行配置和管理：

1. **大语言模型API**: 包括但不限于OpenAI、Anthropic、百度文心、阿里通义、智谱AI、Moonshot等。用户输入的提示词将发送至用户选择的API服务。
2. **云存储服务**: 用户可选择配置对象存储服务（如AWS S3、阿里云OSS）用于数据持久化。
3. **消息推送服务**: 用户可配置钉钉、飞书、企业微信、Slack、Telegram等消息通道用于通知推送。

**重要提示**: 上述第三方服务的使用受其各自的服务条款和隐私政策约束。本软件不对第三方服务的数据处理行为承担任何责任。用户应仔细阅读并理解所使用的第三方服务的隐私政策。

### 2.5 Cookie和追踪

本软件**不使用**Cookie、本地共享对象（Flash Cookie）、Web Beacon、像素标签或任何其他形式的在线追踪技术。

### 2.6 用户权利

根据适用的数据保护法律法规，用户享有以下权利：

1. **查看权**: 用户有权查看本软件在其本地设备上存储的所有数据文件。
2. **删除权**: 用户有权随时删除本软件生成的任何本地数据文件，包括配置文件、日志、缓存及任务历史。
3. **修改权**: 用户有权修改本软件的配置文件和设置。
4. **导出权**: 用户有权将其本地数据导出为通用格式（JSON、CSV等）。
5. **限制处理权**: 用户可通过本软件的配置选项限制数据处理行为（如关闭日志记录、禁用审计功能等）。

用户可通过直接访问本地存储目录或使用本软件提供的命令行工具行使上述权利。

## 5. 版本历史和更新记录

### v1.0.0 (2025-01-01)

- 首次正式发布。

- 发布初始法律声明与隐私政策文档。

### v0.9.0 (2024-10-15)

- Beta版本发布。
- 新增安全模块（aisec），包括对抗鲁棒性、对齐审计、代码安全扫描等功能。
- 新增动作确认守卫（ActionGuard），支持敏感操作拦截与用户审批。
- 新增联邦学习模块，支持多节点模型训练与聚合。

### v0.8.0 (2024-07-01)

- Alpha版本发布。
- 实现多Agent协作框架（Swarm模块）。
- 新增RAG检索增强生成管线。
- 新增机器人控制模块（Robot模块）。

### v0.5.0 (2024-04-01)

- 内部测试版本。
- 完成LLM多供应商集成（OpenAI、Anthropic、DeepSeek、Moonshot等）。
- 实现计算机使用（Computer Use）模块。

### v0.1.0 (2024-01-01)

- 项目初始化。

- 建立基础项目结构与开发规范。
